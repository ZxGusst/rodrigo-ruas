## 6. ANÁLISE DO CTA — COMO O SITE ORIGINAL CONVERTE

### Como funciona hoje no site original
```
Usuário vê pacote → Clica em qualquer botão/imagem → Abre WhatsApp DIRETO
Link: api.whatsapp.com/send?phone=5511966401489
```

**Não há formulário.** O CTA é WhatsApp direto em TODOS os pontos de contato:
- Botão flutuante de WhatsApp na página
- Clique em qualquer pacote → WhatsApp
- Botão de contato → WhatsApp
- Nenhum campo para preencher

### Como está no nosso site (atual)
```
Usuário clica em CTA → Abre modal com formulário → Preenche campos → Envia
```

### ⚠️ Discrepância crítica
O site original usa **fricção zero** — WhatsApp direto. Nosso formulário adiciona etapas. Para o público 60+, o WhatsApp direto pode ser mais eficaz.

**Recomendação:** Discutir se o formulário é uma melhoria intencional (captura de lead + contexto do interesse) ou se devemos manter o fluxo original de WhatsApp direto.
